## Requirements

Gnu Stow is used to install the config files.

## Installation

To install the config files, type:

    > ./install [[package] .. [package]]


Examples:
    > ./install

    > ./install bash

    > ./install git vim

For more specific instructions go to [INSTALL](INSTALL.md).
